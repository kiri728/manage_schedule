# Pokémon Speed Tool（poke-dbweb）

ポケモンの戦闘データを扱うWebアプリ（非公式・ファンプロジェクト）

---

## 概要
このプロジェクトは、ポケモン対戦における素早さの比較や関連データの確認を目的としたWebツールです。  
既存の情報サイトで不便に感じた点を改善しながら、段階的に機能を追加していきます。

---

## 主な機能（予定）
- 素早さステータスの比較
- 努力値・個体値・性格補正を考慮した計算
- 抜き調整の確認（例：最速○○抜き）
- 条件別の比較表示

---

## 使用技術
- HTML
- CSS
- JavaScript

---

## 目的
- Web開発の学習
- ゲームデータの整理・可視化
- 対戦準備の補助ツール作成
